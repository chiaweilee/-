## HTML特殊符号

| HTML | 符号 | 含义 |
| ---- | ---- | ---- |
| &lt; | < | less-than |
| &gt; | > | greater-than |
| &quot; | " | double quotation mark |
| &apos; | " | single quotation mark (apostrophe) |
| &reg; | ® | registered trademark |
| &copy; | © | copyright |
| &trade; | ™ | trademark |
| &nbsp; |   | Non-breaking Space |
